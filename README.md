# OpenVINO-Colab-Inference-Sample-using-various-frameworks-weights
OpenVINOのColaboratory上での推論サンプルです。<br>
以下フレームワークの重みを read_model() で読み込みます。
* OpenVINO
* ONNX
* TensorFlow
* TensorFlow Lite
* PaddlePaddle


# Usage
Colaboratoryでノートブックを開き、上から順に実行してください。<br>
